Top Down Shooter

Participation project for hackathon "Build on Redis Hackathon". Game serves as a Proof of Concept for managing the full game state on redis. Literally built on Redis!

Game is developed on a publicly accessible github repository. Fell free to explore and learn How you could implement your own active-active geo distributed redis database.

Check the code

Active-Active

All database clusters are actively synchronized with each other in a conflict free manner. Each cluster has full context of data.

Geo Distributed

Database clusters are deployed in geographically distinct places ensuring low latency, higher availability and resilience to errors.

Unified Gamestate

Unified game state can be utilized to enable in game trades, markets, messages and even core gameplay without desynchronization.

Data Processing

By utilizing Redis Gears and RediSearch you are able build complex data piplines directly from your database.
